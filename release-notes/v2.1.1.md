# ArtifactKeep v2.1.1 Release Notes

Date: 2026-02-15

## Added

- Export directly from the prompt edit modal for both image prompts and system prompts, so you can save work without leaving the editor.
- Added first-class import support for SillyTavern character card PNGs in both modal import and bulk import flows.

## Improved

- Single-item prompt export is now faster and cleaner:
  - skips unnecessary organization/ZIP steps when exporting one prompt
  - keeps system prompt formatting intact for single `.md`/`.txt` exports
- Character card import flow is more robust and flexible:
  - detects valid character card metadata from PNG files
  - lets you choose how to apply imported data (overwrite title+prompt, append prompt, image-only, or cancel)
  - accepts `.png` files in system prompt import and auto-processes PNG-only batches
- Imported character card images are now copied into app data and thumbnails are generated automatically for prompt cards.
- Internal release workflow/docs were updated to make future releases and release-note generation more consistent.
